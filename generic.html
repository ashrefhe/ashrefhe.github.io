Détection des Transactions Frauduleuses

Projet réalisé chez Attijari Bank, utilisant le machine learning pour identifier automatiquement les opérations bancaires suspectes. Développement en Python (Scikit-learn, Pandas), visualisation avec Streamlit et Power BI.

Vidéo de démonstration

Pour plus de détails : voici le lien GitHub du projet .

Détails techniques

Le système s'appuie sur un jeu de données de transactions réelles (anonymisées). Il applique un prétraitement des données avec Pandas, une détection d’anomalies via Random Forest, et une interface interactive développée avec Streamlit. La visualisation des performances est effectuée via Power BI pour un suivi global des fraudes détectées.
